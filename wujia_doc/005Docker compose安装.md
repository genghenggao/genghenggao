# Docker Compose



- 安装pip

```
sudo apt install python3-pip
```

- 更新一下库

```
sudo apt-get update
```

- 更新一下pip

```
sudo pip3 install --upgrade pip
```

- 安装docker-compose

```
sudo pip3 install docker-compose
```

- 查看docker-compose版本

```
docker-compose --version
```
